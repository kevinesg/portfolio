# kevinesg - Personal Portfolio

This repository contains my personal portfolio website, showcasing my work experience, projects, and hobbies.

## 🚀 Website

Visit my portfolio at [kevinesg.com](https://kevinesg.com)

## 🛠️ Built With

- **Frontend**: React 19, TypeScript, Vite
- **Styling**: TailwindCSS
- **Animation**: Framer Motion
- **Deployment**: Netlify

## 🌟 Features

- Responsive design
- Dark/light theme with smooth transition
- Interactive project showcases
- Integration with external APIs (Chess.com)
- Optimized image loading

## 📂 Project Structure

The main project is in the `portfolio` directory:

```
portfolio/
├── public/            # Static assets
│   ├── _redirects     # Netlify redirect rules
│   ├── robots.txt     # Search engine crawling rules
│   ├── sign.svg       # Site favicon
│   └── assets/        # Images and downloadable files
│       ├── files/     # PDFs and other documents
│       └── photos/    # Website images
│
├── src/               # Source code
│   ├── components/    # React components
│   │   ├── ui/        # UI components (icons)
│   │   └── ...        # Other components
│   ├── data/          # JSON data files
│   ├── lib/           # Utility functions
│   ├── pages/         # Page components
│   ├── App.tsx        # Main application component
│   ├── index.css      # Global styles
│   └── main.tsx       # Application entry point
│
└── ...                # Configuration files (package.json, tsconfig, etc.)
```

## 🚀 Local Development

1. Clone the repository
2. Navigate to the portfolio directory:

```bash
cd portfolio
```

3. Install dependencies:

```bash
npm install

```

4. Start the development server:

```bash
npm run dev
```

5. Open [http://localhost:5173](http://localhost:5173) in your browser
